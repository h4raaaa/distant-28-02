## Введение

**Назначение:** Веб-приложение для быстрой оценки соответствия веса и роста путем расчета индекса массы тела (ИМТ).

**Функции:**
*   Расчет ИМТ по весу (кг) и росту (см).
*   Определение категории веса (дефицит, норма, избыток, ожирение).
*   Цветная шкала-индикатор результата.
*   Сохранение истории 5 последних расчетов.
*   Текстовые рекомендации.


## Примеры использования

| Действие пользователя | Реакция системы |
| :--- | :--- |
| **Ввод роста**| Поле принимает только цифры. |
| **Ввод веса**| Поле принимает только цифры. |
| **Нажатие «Рассчитать ИМТ»** | Проверка полей >0 |
| **После расчета** | Отображение числа ИМТ.|
| **Определение категории** | Вывод текста: «Нормальная масса тела». |
| **Цветовая индикация** | Шкала подсвечивается зеленым (норма), желтым (избыток) и т.д. |
| **Нажатие «Очистить»** | Поля обнуляются, результат скрывается. |
| **Наведение на шкалу** | Всплывающая подсказка с диапазонами значений. |
| **Ввод букв в поле** | Символы не отображаются. |
